# Setting up dev environment to test Sneat apps locally

## Running Google AppEngine app locally

To run Google AppEngine app locally execute:

```shell
./serve_gae.sh
```

## Running Firestore emulators locally:

To run Google AppEngine app locally execute:

```shell
./serve_fb_emulators.sh
```
